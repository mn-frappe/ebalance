# Configuration

## eBalance Settings

Navigate to **eBalance Settings** in Frappe to configure your API credentials.

### Required Fields

| Field | Description |
|-------|-------------|
| API Key | MOF eBalance API key |
| Company Registration | Company registration number |
| Environment | Production or Sandbox |

### Chart of Accounts Mapping

eBalance requires mapping your ERPNext Chart of Accounts to MOF standard accounts.

1. Download the mn_chart.csv template
2. Map your accounts to MOF account codes
3. Upload the mapping file

## Testing Connection

Use the **Test Connection** button in eBalance Settings to verify your credentials.

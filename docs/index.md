# eBalance - MOF Financial Reporting

Mongolia Ministry of Finance (Сангийн яам) eBalance Financial Reporting System Integration for ERPNext.

## Features

- **100% MOF API Integration** - All eBalance endpoints implemented
- **Financial Statements** - Balance Sheet, P&L, Cash Flow
- **Chart of Accounts Mapping** - mn_chart.csv compliance
- **Multi-Company Support** - Handle multiple entities

## Quick Start

```bash
bench get-app https://github.com/mn-frappe/ebalance
bench --site your-site.local install-app ebalance
```
